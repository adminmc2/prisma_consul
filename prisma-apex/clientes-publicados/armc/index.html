Metodología APEX

¿Cómo se lee este análisis?

Cada diagrama muestra el flujo actual (as-is) basado en las entrevistas operativas realizadas.

Elementos clave:

  • Fases — Etapas del proceso (ej: Captación → Conversión → Seguimiento)
  • Handoffs — Puntos de transferencia entre roles
  • Fricciones — Cuellos de botella identificados
  • Herramientas — Sistemas usados (Excel, WhatsApp, papel, etc.)
  • Gaps — Información que no fluye correctamente

Los 4 Dolores (D1-D4)

Cada fricción se mapea a uno de los 4 dolores empresariales:

  • D1 — Procesos Internos: Falta de estandarización, duplicación
  • D2 — Seguimiento y Clasificación: Pérdida de información, trazabilidad
  • D3 — Comunicación Personalizada: Falta de contexto en handoffs
  • D4 — Conversión de Leads: Velocidad, conversión, recurrencia
